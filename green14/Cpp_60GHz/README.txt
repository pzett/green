First use in a computer:
"make clean" in the "tx_60GHz" and "rx_60GHz" in every folder used
"make tx_60GHz" in every used folder
"make rx_60GHz" in every used folder

In both computer:
Connect the USRP.
uhd_find_devices command in order for the computer to find the USRPs.
Open terminal command line

On the transmiter computer:
Open a tx folder
see specific read me

On the receiver computer:
Open a rx folder
see specific read me



First, run rx so that it will wait for a reception

Second, run the transmittion and wait for the reception


Wait for the decoding and the shown results

Whenever the transmitter or the receiver ask for the sudo code, enter your sudo code.
