Receiver code for the demonstration of QPSK transmission. (Usually 0 BER)

Quick version without Kalman or system of Queues

Short Doc how to use the code:

- Compile with command: make

- run with command: sudo ./rx_60GHz

- Wait for the data to be received

- Octave will display the received data, press any key to continue

- Be careful about the receiver power threshold
